# VR_interactable_Solar_System_with_Vuforia
This is a interactable solar system build with the stack of Unity+Vuforia AR engine where Vuforia hold all the VR camera and detection function while Unity helps on interactable logic, collision detect, triggered and displaying 3D models, description and more
